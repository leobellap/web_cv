Projects:

1. Tool for Analyzing Political Trends on Reddit

Developed a tool to extract and analyze top and trending topics from the r/politics subreddit. This tool was used to gather data several weeks before and after the US 2024 elections. By visualizing this data, it is possible to track how political discussions and priorities evolved throughout the election period.

Key features:

  • Developed a method to efficiently parse and structure the data for analysis.
  • Saved the data as CSV files throughout the research period.
  • Implemented a robust data cleaning pipeline to remove HTML tags, links, special symbols, and other noise.
  • Selected an optimal embedding model to ensure accurate sentence embeddings.
  • Trained model and experimented with various hyperparameters to achieve the best performance.
  • Created visualizations to illustrate how topics changed and evolved during the election period.

2. Web Application for Evaluating Process Models

Developed a comprehensive web application designed to gather and analyze human perceptions of process models. Participants provide demographic information and subsequently rate various process models, enabling in depth studies on model interpretability and ideal metrics for evaluating model complexity and understandability. The key part of this project is tracking a user's mouse movement and then saving it to a database for a further research.

Example of model rating process

Key features:

  • Developed and designed the frontend with HTML, Tailwind CSS, JavaScript and Figma.
  • Created a mouse tracking script with JavaScript to track the movement of the handle of a 2D slider.
  • Implemented PostgreSQL database, created necessary tables.
  • Utilized Python, SQLAlchemy and Flask for backend development.
  • Containerized the application using Docker.
  • Configured Nginx server and hosted project on Yandex.Cloud.
  • Integrated Yandex.Metrica for detailed user interaction tracking.
